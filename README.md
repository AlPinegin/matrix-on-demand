# Matrix On Demand

A research and creative project about the evolution from search engines and
recommendation systems to personalized reality generation.

The central idea:

> Search became recommendation.
> Recommendation became advertising.
> Advertising became music.
> Music became experience.
> Experience became reality.

Matrix On Demand treats an album as a chain of discoveries. Each track is not
only a song, but an argument, a scene, and a hook into the next idea.

## Repository Shape

- `FOUNDING.md` records the origin of the project.
- `docs/manifesto.md` defines the philosophical frame.
- `hooks/hooks.md` collects lyrical and conceptual hooks.
- `albums/matrix-on-demand/` will hold the album structure, tracks, prompts,
  lyrics, notes, and production direction.

## Working Principle

This repository is not just storage for song drafts. It is an idea lab.

Every meaningful change should answer one of these questions:

- What did we discover?
- What changed in our understanding?
- What new hypothesis appeared?
- What hook is strong enough to become a song?
